## 
CPU-based raytracer following Peter Shirley's book. Future plans include CUDA optimization for GPU acceleration.


## Usage Guide

### Clone the repo

```
git clone git@github.com:lucascogrossi/raytracer.git
```
### Compile main.cpp
```
g++ main.cpp
```

### Run
```
./a.out > image.ppm

eog image.ppm
```
